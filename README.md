이 저장소는 인공지능 기반의 주식 추천 대시보드부터 자동차 레이싱 게임까지, 다양한 웹 애플리케이션 모음입니다. 별도의 설치 없이 브라우저에서 바로 실행 가능한 단일 HTML 파일 형태로 제작되었습니다.

📂 포함된 프로젝트

1. 📈 AI 주식 추천 대시보드 (stock_recommend_ai.html)

설명: 국내 저평가 우량주 20선을 엄선하여 제공하는 투자 보조 도구입니다.

주요 기능:

정적 데이터 제공: AI가 분석한 기업별 상세 설명 및 투자 포인트 포함.

페이지네이션: 20개 종목을 10개씩 나누어 탐색 가능.

실시간 차트: TradingView 위젯을 연동하여 실시간 주가 흐름 확인.

종목 상세 카드: 클릭 시 기업 분석 데이터와 차트가 동기화되어 표시.

2. ✅ 심플 투두 리스트 (todo_app.html)

설명: 깔끔한 디자인의 할 일 관리 애플리케이션입니다.

주요 기능: 할 일 추가/삭제, 완료 체크, 남은 항목 카운트, 완료 항목 일괄 삭제.

3. 🏎️ 터보 레이서 - 자동차 게임 (car_racing_game.html)

설명: HTML5 Canvas를 이용한 3D 가상 레이싱 게임입니다.

주요 기능: 키보드 및 모바일 터치 컨트롤 지원, 속도 기반 난이도 조절, 장애물 회피 로직.

🚀 실행 방법

이 저장소를 클론(Clone)하거나 원하는 HTML 파일을 다운로드합니다.

웹 브라우저(Chrome, Edge, Safari 등)에서 해당 파일을 더블 클릭하여 실행합니다.

🛠️ 기술 스택

Frontend: HTML5, Tailwind CSS, JavaScript (Vanilla JS)

External APIs: TradingView Widget (Stock Charts)

Design: Font Awesome Icons, Google Fonts (Pretendard)

📝 참고 사항

본 앱의 주식 추천 데이터는 정적 데이터로, 실제 투자 시에는 최신 재무 정보를 직접 확인해야 합니다.

모든 앱은 단일 파일(Single File) 구조로 설계되어 배포 및 공유가 간편합니다.
